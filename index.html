Introduction to Python

Python is a popular, high-level, interpreted programming language known for its simple syntax and readability.

It was created by Guido van Rossum and first released in 1991.

Python can be used for web development, data analysis, artificial intelligence, and more.

  • Easy to read and write
  • Interpreted language
  • Supports multiple programming paradigms
print("Hello, World!")
Variables and Data Types

Variables in Python do not require explicit declaration to reserve memory space.

The data type is inferred from the value assigned to the variable.

  • int
  • float
  • str
  • bool
  • list, tuple, dict, set
x = 5 # int
name = "Alice" # str pi = 3.14 # float
Control Flow

Control flow statements allow you to execute code blocks conditionally or repeatedly.

The most commonly used statements are if, for, and while.

  • Conditional statements (if, elif, else)
  • Loops (for, while)
  • break and continue
if x > 0:
  print("x is positive") for i in range(5):
  print(i)
Functions

Functions are reusable blocks of code that perform a specific task.

You can define a function using the def keyword.

  • Define a function with def
  • Use parameters and return values
  • Functions can have default arguments
def greet(name):
  print(f"Hello, {name}!")
Modules and Packages

Modules are files containing Python code, and packages are collections of modules.

You can import modules using the import statement.

  • Import standard library modules
  • Create your own modules
  • Install packages with pip
import math
print(math.sqrt(16))
